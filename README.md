# Game-based-learning
Developers: Cuma Selçuk Kulaksız Roles: scripter, ui designer, level designer, audio designer( not the audio artist)
Project Name: Axiom Weavers
Axiom Weavers" is a turn-based tactical puzzle game set in a universe where mathematical concepts form the physical fabric of reality. The "Prime Continuum" is under attack by glitches known as "Paradoxes." The player's primary goal is to command a "Scholar" on a 2D coordinate grid, utilizing geometric strikes to defeat the Paradoxes and restore logic to the battlefield. The game blends the cognitive engagement of educational mathematics with strategic positioning, transforming Cartesian coordinate navigation into the core combat mechanic rather than relying on traditional random number generation (RNG) or simple proximity attacks.

Educational Concept
Pedagogical Goal: The primary educational goal is to teach players Cartesian Coordinate Geometry (navigating an X, Y plane) and foundational spatial awareness.

Applied Learning Theory:
The game utilizes Experiential Learning Theory (learning by doing) and Constructivism. Instead of passively reading a textbook about how X and Y axes intersect, the player actively constructs their understanding of the grid by testing mathematical inputs and experiencing immediate visual consequences on the battlefield.

How Game Mechanics Facilitate Learning:
The core combat mechanic requires the player to target enemies using specific grid coordinates. If an enemy is standing on a specific tile, the player cannot simply click on the enemy to attack them. Instead, the player's "Coordinate Strike" ability requires them to correctly identify and manually input the intersection of the X-axis (horizontal) and Y-axis (vertical) through the User Interface. A correct input results in a successful hit on the enemy (positive reinforcement), while an incorrect input wastes an Action Point and misses the target. This immediate, high-stakes feedback loop forces the player to carefully calculate and internalize spatial positioning before executing an action.
